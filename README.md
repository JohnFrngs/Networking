# Cisco Networking Labs — MSc Informatics, University of Piraeus

Two progressive networking labs implemented in Cisco Packet Tracer,
covering static and dynamic routing, NAT, and access control.

| Feature                  | Lab 1 — Static Routing | Lab 2 — OSPF + NAT + ACL |
|--------------------------|------------------------|--------------------------|
| Routing protocol         | Static                 | OSPF (Area 0)            |
| Subnet types             | /29, /30               | /19, /30, /24            |
| Internet simulation      | ✗                      | ✅ (ISP + DNS)            |
| NAT                      | ✗                      | ✅ PAT/Overload           |
| Access control           | ✗                      | ✅ Extended ACLs          |
| Routers                  | 3                      | 3 + ISP router           |
